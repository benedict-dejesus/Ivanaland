# IVANALAND — 09 PHONE BIBLE

`src/data/phones.ts` (single source of truth for the engine). Format per phone:

> **#ID — "Name"** · District · Landmark · Host object
> Category / Tier / Discovery method
> Story — why it's there. Hint — what the player reads.

Categories: visible, camouflaged, partial, container, reflection, interaction,
sequential, legendary. Tiers 1–5 per 08_HIDING_TAXONOMY. Methods: `tap`,
`container`, `reveal`, `sequence`.
